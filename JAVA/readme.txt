一、光盘内容

    本光盘是《深入体验Java项目开发》一书的配书光盘，内容包括书中所有案例的视频讲解、项目源程序代码以及赠送的十个案例源代码。

    本光盘根目录下有3个文件夹，1个readme文件、1个赠送案例的说明文件。

二、运行环境

    1、硬件最低要求 
    C   P  U：Pentium（II）；
    内    存：256MB；
    硬盘空间：500MB（可选）；
    显示模式：1024*768，16位增强色；
    其    他：标准键盘、鼠标、声卡、音箱。

    2、软件环境 
    软件支持：暴风影音3.11.04.06及以上版本 

    本书代码均在Eclipse SDK Version: 3.6.0调试通过，将软件JDK安装在D:\Program Files\Java ，将Eclipse软件安装在D:\Program Files\eclipse，将MySQL软件安装在D:\Program Files\MySQL,然后建立MySQL用户名是root, MySQL密码是rootroot,导入Hibernate3包；在D盘根目录建立java目录
